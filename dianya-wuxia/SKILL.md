# 典雅白话武侠引擎

**专为武侠/江湖题材设计的写作系统，基于白话江湖风格实践验证。**
> 先加载 `dianya-novel-engine` 获取共享风格规则，再执行本引擎。本引擎与`dianya-engine-short/medium/long`配合使用——武侠题材先走主引擎 Phase 1 确定篇幅结构，再加载本引擎执行武侠专项技法。

---

## 1. 武侠与典雅白话的关系

武侠小说天然适合典雅白话。原因：古典背景屏蔽了现代事物（天然过"出今禁令"），门派/兵器/招式自带古典词汇场，江湖道义提供了天然的价值张力。

但武侠的"古风浓度"应低于纯文学古典小说。原因：武侠读者需要的是**快节奏**，过高的古典词密度会拖慢阅读速度，与武侠的"快意恩仇"气质冲突。

**武侠的古风配方**（与纯文学古风的区别）：

| 维度 | 纯文学典雅白话 | 武侠典雅白话 |
|------|-------------|------------|
| 古典词密度 | 5-15% | 3-8% |
| 段落长度 | 一句一段（松） | 一句一段（更紧，节奏突刺） |
| 对话占比 | 25-35% | 40-55% |
| 叙述者声音 | 限知第三人称为主 | **评论式第三人称**（反问叙事） |
| 物象系统 | 核心技法 | 极少使用——动作和对话承担情绪 |
| 古典感来源 | 古词+意象 | **句式节奏**（对仗/四字格/反问/短句） |

## 2. 武侠叙事声音

### 评论式第三人称：反问叙事

武侠叙述不是隐身的。叙述者偶尔跳出来，用一句反问品评世事——然后立刻退回场景。

**操作规则**：
- 每章反问不超过 **3 处**
- 反问只出现在**情绪高点或段落收束处**
- 反问之后立即切回事件叙述
- 反问句格式：岂非……？/ 何尝不是……？/ 怎的……？/ 谁又……呢？

> 示例（来自白话江湖）：
> "人生会遇上许多的事，可是真正让人轻松而开心的事，岂非都是一些小事？"
> ——反问停在恰好。下一段切回场景："三人一路乘着春风暖日，一路闲聊走向春风得意楼。"

> "好心办坏事岂非是生活中比较常见的事？"
> "千金易得，知己难寻。"
> "世事岂能尽如人意？"

**与说书人腔的区别**：说书人腔用"话说/且说"控制叙事流向。反问叙事用疑问句引发读者共鸣。前者是操控节奏，后者是邀请共情。

### 叙述者戒律

- ✅ 反问后立即切回场景——不展开议论
- ✅ 反问句只放在段尾或场景切换处
- ❌ 反问句后不接"因为……所以……"的解释
- ❌ 两个反问不连续出现——反问之间至少要隔 500 字

## 3. 武侠结构

### 短篇武侠（8千-3万字）

**一剑封喉**。一个核心战斗场景撑一篇。

```
开篇（招式亮相）→ 铺垫（恩怨建立）→ 升级（冲突加剧）→ 决战（最强一招）→ 余韵（收剑）
```

| 段 | 占比 | 功能 | 武侠专属要点 |
|----|------|------|------------|
| 开篇 | 5-10% | 亮出武功或刀剑 | 用动作或兵器吸引注意力，不写天气 |
| 铺垫 | 30-40% | 恩怨的来龙去脉 | 用对话交代，不用回忆杀 |
| 升级 | 20-30% | 小战或对峙 | 先展示对手的实力，为决战蓄势 |
| 决战 | 10-15% | 最强招式的释放 | 快速拆招→慢写绝技→一击收场 |
| 余韵 | 5-10% | 收剑转身 | 金句收束，不解释不感慨 |

### 短篇武侠写作原则

- **开篇亮兵器**：前 300 字内必须出现武功/兵器/打斗——不要让读者等
- **恩怨一句话说清**：短篇没空间铺垫复杂的门派历史。恩怨用一句台词交代——"你杀了我师兄。""是。"
- **只写一场完整的战斗**：可以有对峙/小冲突，但全篇只有一场从头写到尾的完整打斗——就是决战
- **绝技留到最后**：主角最强的招式不在开篇展示。决战时才放——放了就必须结束战斗
- **结尾收剑**：打完即止。不写"从此以后"、不写角色去向、不总结恩怨

### 短篇武侠常见失败模式

| 失败模式 | 症状 | 修复 |
|---------|------|------|
| **开篇没拔剑** | 前500字在写风景/背景/人物外貌 | 把决战中的一句动作挪到开篇 |
| **恩怨太复杂** | 用了半篇解释门派历史 | 恩怨浓缩到两个角色之间的私人仇恨——师门/家族级别的恩怨留给中长篇 |
| **打斗太啰嗦** | 一场打斗写了3000字还没结束 | 短篇打斗控制在1000字以内。快招速写为主，绝技一击收场 |
| **反派太弱** | 读者感觉主角随便就能赢 | 决战前让反派先赢一个小回合——展示他的实力 |
| **打完还聊** | 决战之后角色开始总结人生 | 删。决战的最后一击就是结尾。加一句金句收束，全文结束。 |

### 短篇武侠输出范例

以下是一段短篇武侠开篇（8000字短篇的开头）：

```
剑是从窗子进来的。

沈九正在喝茶。茶盏举到嘴边的时候，窗纸破了一个洞，一道青光穿过来，钉在身后的柱子上。是一把没有剑穗的长剑，剑身还在颤。

沈九没有回头。他把茶喝完，放下杯子，拔下柱子上的剑，走到窗前。

外面没有人。巷子里只有一条黄狗在墙角趴着，看到沈九，摇了摇尾巴。

剑身上刻了一个字。

"仇。"

沈九看了看那个字，把剑横在桌上，坐回椅子，继续倒茶。他倒了两杯。

一盏茶之后，巷子里有了脚步声。
```

### 中篇武侠（3万-10万字）

**江湖恩怨**。完整起承转合，有门派冲突和 1-2 条副线。

```
起（结识/入局）→ 承（联手/冲突展开）→ 转（背叛/真相）→ 合（决战/收束）
```

每章核心事件中至少包含：
- 一次**对话交锋**（斗嘴/道义辩论/密谈）
- 一处**武功展示**（不一定是战斗，可以是轻功/身法/内力）
- 一个**新信息**（让读者对角色或江湖了解更多）

### 中篇武侠写作原则

- **每章至少一次剑出鞘**：不需要每章都有打斗，但每章都要让读者感觉到"武功就在手边"——拔剑/运气/轻功/暗器，动作随时可能发生
- **门派冲突在转折点爆发**：起点展示门派关系，承段深化矛盾，转段让门派正面冲突——一场关键对决或一次背叛
- **反派层次**：中篇的反派不能只是"坏人"。他有自己的门派归属、师父、要保护的人。他的行为和主角一样有逻辑
- **武功升级有代价**：中篇中主角武功可以进步——但每次进步必须经历失败或受伤。不能"悟了一下就突破了"
- **章尾悬念三种武侠模式**：要么一个对手出现、要么一个秘密被揭露、要么一把剑出鞘

### 中篇武侠武功追踪（轻量版）

中篇也需要武功追踪，但比长篇简单——不需要全套武功体系文档，仅需一个表：

```
| 角色 | 标志武功 | 第N章状态 | 是否受伤影响武功 |
|------|---------|----------|---------------|
| 风满堂 | 无定式（应变型） | 正常 | 否 |
| 柳涣歌 | 六十四卦剑 | 正常 | 否 |
| 百里一战 | 参商双剑 | 一梦黄粱未熟 | 否 |
```

每写满 5 章更新一次即可。

### 中篇武侠常见失败模式

| 失败模式 | 症状 | 修复 |
|---------|------|------|
| **江湖太大，篇幅太小** | 写了5个门派但每个都没深度 | 核心门派限制2-3个。其他门派一句话带过。 |
| **战斗密度过高** | 每章都有完整打斗，读着累 | 中篇每2-3章一次完整打斗。用对峙/口舌之争/展示轻功来填充。 |
| **副线压主线** | 副线角色比主线更有意思，读者"歪了" | 如果副线确实更好→切过去当主线。如果副线只是抢戏→砍掉副线独立场景，只在主线场景中让副线角色出现。 |
| **转折不是武功导致的** | 转折点是一个误会/巧合/外人介入 | 武侠的转折应该由武功来驱动——一场战斗的胜负改变局势、一招失手暴露身份。 |
| **招式名通货膨胀** | 同一角色用了七八个不同招式名 | 中篇每个角色2-3个招式名足够。更多的时间用在"同一招式在不同局面下的变化用法"上。 |

### 中篇武侠输出范例

以下是一段中篇武侠开篇（起·第一章片段）：

```
风满堂是被一只麻雀吵醒的。

"风大哥风大哥风大哥！"

他闭着眼睛翻了个身。麻雀跳到他的被子上，继续叫。

"你再不起来我就去告诉白衣哥哥你昨晚又偷喝酒了！"

风满堂睁开一只眼，看见欧阳欢欢蹲在床沿，两只大眼睛离他的脸只有三寸。他叹了口气。

"你说吧。反正他打不过我。"

"那可不一定，我白衣哥哥最近剑法进步好多好多，说不定就把你打趴下了！"

风满堂坐起来，拿起床头的酒葫芦灌了一口。凉酒入喉，人清醒了几分。窗外是洛阳四月的阳光，暖得让人什么都不想做。

"有什么事？"

"城外来了三个人。"欧阳欢欢的表情难得正经了一点，"一个背着剑，一个拿着扇子，还有一个……在河里睡觉。"

风满堂放下酒葫芦。这三个人的组合听起来不像是来找麻烦的。但三个人一起出现，从来不会没有麻烦。

"带我去看看。"

他抓起自己的刀。刀很轻，出鞘的声音很细。欧阳欢欢说那声音像女人的叹息。风满堂觉得更像一声没有说出口的警告。
```

### 长篇武侠（10万字+）

**武林争霸**。多卷、多势力、多线交织。

武侠长篇必须先遵循 `dianya-engine-long` 的全部基本规范（治理文档/追踪文档/连贯性保证/阶段审计/字数预算/POV/变更管理/崩溃急救），再叠加以下武侠专属规范。

### 长篇武侠专属治理文档

在主引擎长篇治理文件的基础上，武侠长篇额外维护：

```
项目/
├── 武侠治理/              # 武侠专属治理文档
│   ├── 武功体系.md         # 核心武功层级/克制关系/修炼条件/代价
│   ├── 势力格局.md         # 各卷势力分布与变化（联盟/敌对/中立）
│   ├── 角色武功追踪.md     # 每个主要角色当前武功境界/已掌握招式
│   └── 卷级战斗规划.md     # 每卷安排几次关键战斗、战斗目的
```

### 武功体系文档格式

每新增一门武功即追加：

```
| 武功名称 | 层级 | 所属门派/人物 | 克制什么 | 被什么克制 | 代价/限制 | 首次出现章节 |
|---------|------|------------|---------|----------|----------|------------|
| 灵汐柔水 | 门派绝学 | 归墟剑派汐脉 | 刚猛型武功 | 以快破柔 | 需借力化力，对极快武功被动 | 第2章 |
| 情花剑雨 | 镇派之宝 | 薛残阳/薛弘毅 | 群战 | 幽魂手 | 消耗内力极大，使用后虚弱三日 | 第5章 |
```

**武功体系铁则**：
- 任何"镇派之宝"级别的武功必须在第一卷内展示其**弱点或代价**——读者不能到了第五卷才发现"原来这招有破绽"
- 同一层级武功之间必须有明确的克制关系（A克B，B克C，C克A），避免"谁内力强谁赢"
- 主角禁止在最后一卷之前学会"传说境界"的武功——最强招留到最后
- 每卷结束时更新角色武功追踪——谁突破了、谁受了重伤影响武功、谁获得了新招式

### 角色武功追踪格式

```
| 角色 | 当前境界 | 已掌握招式 | 正在修炼 | 武功状态 | 上次更新卷章 |
|------|---------|----------|---------|---------|------------|
| 柳涣歌 | 一流高手 | 六十四卦剑、天地同心 | 无极心法 | 正常 | 第2卷第5章 |
| 百里一战 | 一流高手 | 参商双剑、一梦黄粱 | 一梦黄粱未纯熟 | 正常 | 第2卷第5章 |
```

### 势力格局文档格式

每卷开始前确定，每卷结束后更新：

```
第N卷势力格局：
- 正派联盟：{成员} / 关系：{紧密/松散} / 变化：{上卷→本卷}
- 邪派势力：{成员} / 威胁等级：{高/中/低}
- 中立势力：{成员} / 倾向：{偏正/偏邪/真中立}
- 本卷格局变化目标：{旧联盟破裂/新势力崛起/排名重洗/……}
```

### 卷级战斗规划

长篇武侠最容易出现的崩坏是"战斗疲劳"——读者到了第四卷已经看腻了打架。因此每卷的战斗必须有**功能分工**：

| 卷 | 战斗功能 | 战斗场次 | 关键战斗目的 |
|----|---------|---------|------------|
| 第1卷 | 展示武功世界 | 2-3场 | 让读者知道这个世界的武力是什么样的 |
| 第2卷 | 升级冲突 | 3-4场 | 主角遇到更强的对手，发现自己还不够强 |
| 第3卷 | 转折之战 | 2-3场 | 一场大战改变势力格局（正派联盟分裂/主角受重创/关键人物死亡） |
| 最后一卷 | 终极对决 | 1-2场 | 最高武力的碰撞，解决全书核心矛盾 |

**战斗密度红线**：
- 不允许连续3章都有完整打斗——每一场战斗之后至少用1章缓一口气
- 每卷至少有一场战斗是**不打完**的（中断/撤退/自然化解）——给读者"没看够"的感觉
- 长篇的终极对决不能早于倒数第3章——最后2章用来收束情感和角色去向

### 长篇武侠阶段审计（额外项）

在 `dianya-engine-long` 的阶段审计基础上，武侠长篇增加：

| 审计项 | 触发条件 | 检查内容 |
|--------|---------|---------|
| 武功一致性 | 每10章 | 角色的武功境界与上次记录一致？新学的招式登记了？ |
| 势力格局检查 | 每卷结束 | 势力格局是否按计划变化？是否需要调整下卷格局？ |
| 力量膨胀预警 | 每20章 | 主角武功是否增长过快？反派的威胁是否跟得上？ |
| 战斗疲劳检查 | 每10章 | 最近10章打斗占比是否超过30%？是否有连续3章都有完整打斗？ |

### 长篇武侠专属崩坏预防

| 崩坏类型 | 症状 | 预防 |
|---------|------|------|
| 力量膨胀 | 主角每卷升一级，第六卷已天下无敌 | 武功体系文档设定好上限。升一级要有代价（受伤/失去/代价）。 |
| 战斗疲劳 | 读者说"又打了" | 战斗密度红线：不连续3章打斗。每打一场缓一章。 |
| 势力过多 | 读者记不住谁是谁 | 核心势力不超过5个。其他势力一笔带过。每卷势力格局变化只在核心势力之间发生。 |
| 武功通胀 | 第一卷"剑气"就很厉害，第五卷"剑气"变杂兵技能 | 武功体系建立时就定好层级。杂兵永远用不到"门派绝学"及以上。 |
| 反派降智 | 前期很聪明的反派后期变蠢 | 每个反派都有一个"不蠢的原因"——他被什么限制了？信息不足/内部分裂/被更高层操控？ |
| 招式遗忘 | 主角第一卷学的绝招后来再没用过 | 每次战斗前查角色武功追踪表。该用的老招要安排用上——用出新意才是好武侠。 |

## 4. 武侠角色系统

### 角色类型与对话模式

| 原型 | 用词特征 | 对话示例 |
|------|---------|---------|
| **游侠/浪子** | 话少、句短、偶尔调侃，用反问代替正面回答 | "走不走？一句话。" |
| **剑客/武者** | 涉及武功时用词精准，日常对话简练 | "你的剑慢了。""我知道。" |
| **掌门/帮主** | 话有分量，用"本座/本门"，语气居高但不粗暴 | "此事关系我门派兴衰，诸位自行斟酌。" |
| **隐士/高人** | 话少而含蓄，爱说半句留半句，偶尔吟诗引用 | "机缘到了，你自然知道。" |
| **商人/掌柜** | 圆滑市井，时而奉承时而精算，话里有生意经 | "这位爷，咱们做生意的，讲究个和气生财。" |
| **捕快/官差** | 话带规矩，用"大人/本官"，但私下也会放松 | "王法在上，还请诸位给个面子。" |
| **魔头/反派** | 狂傲或冷静，动机说得很清楚，不遮掩自己的目的 | "本座要的东西，还没有拿不到的。" |

### 角色绝技/招牌

每个主要武侠角色至少有一项**让人记住的特质**：

| 类型 | 示例 |
|------|------|
| 武功绝技 | "他的参星商星双剑，让人生不出躲避的念头" |
| 行为习惯 | "他每次出剑之前，先把左手的袖子挽一圈" |
| 口头禅/说话方式 | 结巴、爱反问、说话时摆弄物件 |
| 外貌特征 | 两鬓花白的年轻人、面纱遮脸的剑客 |

### 武侠角色命名

- 姓+名：两字或三字均可，避免"雪儿/灵儿"类昵称作正式名
- 名号/外号：在江湖上比真名更重要——"洛阳小霸王""半日神仙老书虫""沧海剑仙"
- 门派+排行：归墟剑派第三代汐脉弟子、神刀门门主

## 5. 武功描写

### 招式命名规则

| 风格 | 结构 | 示例 |
|------|------|------|
| **意象式** | 物象+动作 | 潮汐剑意、回风游絮、情花剑雨 |
| **典故式** | 成语/诗句化用 | 天地同心、否极泰来、一梦黄粱 |
| **数字式** | 数字+意境 | 六十四卦剑、九字诀、沧海九变 |
| **具象式** | 动作描述 | 参星商星（双剑名）、蜻蜓点水 |

**命名铁则**：招式名要有画面感——读者看到名字能想象出动作。全篇招式命名风格统一。

### 打斗场景节奏

武侠打斗不是武术教材，不需要拆解每一招。**快招速写、绝技慢写**。

```
快招速写（80%的打斗篇幅）：
"二人剑来剑往，已过了五十余招。"
"她剑势一变，连出三剑，一剑比一剑快。"

慢写绝技（关键一击时）：
"百里一战双剑交替刺出，参星刺'伏兔'，商星刺'髀关'。柳涣歌撤脚，收剑斜点百里一战'云门'、'中府'，使出一招'否极泰来'。"
```

**打斗描写禁忌**：
- ❌ 不要逐招拆解（"他出左拳→对方右闪→他变肘击→对方后仰"）——这是武术教材
- ❌ 不要在打斗中大段插入回忆或解说
- ❌ 不要用比喻打断打斗节奏（"他的剑如流星划过夜空"——打斗正紧张，读者不想看星星）
- ✅ 快招用概括（"过了五十余招"），绝技用具体（写出招式名和动作意图）

### 武功体系设计（长篇专用）

长篇必须维护武功层级关系：

| 层级 | 说明 | 示例 |
|------|------|------|
| 基础武学 | 各门派通用的轻功/内功/拳脚基础 | 归墟剑派三脉基础剑法 |
| 门派绝学 | 每派 1-3 项独门武功 | 灵汐柔水（汐脉绝技）、海潮崩云（潮脉绝技） |
| 镇派之宝 | 每派最高武学，极少外传 | 归墟一剑、情花剑雨 |
| 传说境界 | 全书最高武力，几乎不出手 | 沧海剑仙、天地同心 |

**力量平衡铁则**：
- 每项强力武功必须有**代价或限制**（内力消耗/使用次数/反噬风险/修炼条件）
- 主角不能同时拥有两项镇派之宝——一项就够
- 反派的武功不能弱于主角——势均力敌才有好看的打斗

## 6. 武侠对话

### 对话驱动情节

武侠小说中对话占比可达 40-55%。对话承担三重功能：
1. **信息传递**：恩怨线索、江湖规矩、武功秘密
2. **性格展示**：斗嘴/挑衅/试探/道义辩论
3. **节奏控制**：紧张场景的快问快答、酒馆聊天的闲笔对话

### 经典对话模式

**斗嘴式**（最快建立角色关系）：
> "你这人何故多管闲事，扰我清梦？"
> "我见你落水，好心救了你，你反倒怨我多管闲事？"
> "我几时请你救过？我又几时喊过救命？"

**挑衅式**（战斗前奏）：
> "柳涣歌，可敢出城一战？"

**哲理辩论式**（长篇专属，提升作品厚度）：
> "何为天命？何谓常道？"
> "天命，乃先天所赋万物初始之名状……"

**密谈式**（推动暗线）：
> "门主，一切都在按计划进行。"

### 对话中的江湖切口

| 场合 | 说法 |
|------|------|
| 见面礼 | "久仰""幸会""阁下""这位朋友" |
| 挑战 | "请教阁下高招""可敢一战""请赐教" |
| 认输 | "在下技不如人""甘拜下风""领教了" |
| 威胁 | "刀剑无眼""后果自负""休怪在下不讲情面" |
| 道义 | "行走江湖""人在江湖""义字当头" |
| 告别 | "后会有期""就此别过""有缘再见" |

## 7. 江湖世界观

### 江湖道义的默认规则

读者默认角色遵循以下规则（除非你明确写他打破了）：

- **挑战光明正大**：下毒/暗器/偷袭是"下三滥"手段，用则说明角色非正道
- **恩怨分明**：有仇必报、有恩必还。恩怨是武侠最基础的驱动引擎
- **不杀无辜**：即使是反派，杀无辜妇孺也会降低格调
- **技不如人认栽**：公平比试输了就是输了，耍赖比输更丢人
- **门派荣辱重于个人**：师门的名声比个人得失更重

### 正邪之分

武侠的正邪不是绝对的。好武侠的"正"有局限，"邪"有理由：

| 维度 | 正派 | 邪派 |
|------|------|------|
| 行事方式 | 讲规矩、公开挑战 | 不择手段、暗中行动 |
| 目标 | 维护门派/道义/秩序 | 追求力量/财富/复仇 |
| 底线 | 不杀无辜（至少在明面上） | 为达目的可牺牲他人 |
| 读者为什么共情 | 他们有不能违背的原则 | 他们有被伤害的过去 |

**最重要的原则**：反派不是疯子。他做什么事都有他自己的理由——这个理由在他的价值观里是成立的。

## 9. 武侠常见失败模式

| 失败模式 | 症状 | 修复 |
|---------|------|------|
| **打架太多忘了讲故事** | 连续三章都有打斗但情节无推进 | 每场战斗必须改变角色关系或势力格局。打完什么都没变→删掉。 |
| **武功写成了武术教材** | "左拳击面→右闪→变肘→后仰→旋风腿" | 快招速写。除非全书最重要的对决，否则不逐招拆解。 |
| **力量失控** | 主角第五卷已经天下无敌 | 第一卷就设好武功体系的上限和代价。每项强力武功有明确弱点。 |
| **正邪太分明** | 正派全是好人、邪派全是疯子 | 正派角色做一件自私的事。反派角色给一个读者能理解的理由。 |
| **对话不像武侠** | 角色说话像现代都市人 | 用江湖切口。书生说书面语，市井说短句。少用"好的""知道了"。 |
| **章尾没有牵引力** | 读者放下来就不想再拿起来 | 每章结尾至少留：一个未答的问题/一个即将发生的冲突/一句金句。 |

## 11. 古龙风格模式

**古龙（1938-1985），武侠小说史上最具辨识度的风格家。** 他的文字将武侠从"讲什么故事"推进到"怎么讲故事"——极短段落、警句判断、瞬间美学、对话驱动，自成一体。本模式提取古龙风格的核心技法，既可整体切换，也可逐项拆解混搭。

### 风格模式概览

古龙风格模式是对现有武侠引擎的**风格叠加层**——不改动结构框架（短/中/长篇结构不变），但系统性地替换以下维度：

| 维度 | 默认武侠（白话江湖风） | 古龙模式 | 是否可独立调用 |
|------|-----------------|---------|------------|
| 叙述者声音 | 反问叙事（岂非/何尝） | **警句断言**（"人生本就……"） | ✅ 可混搭 |
| 打斗描写 | 快招80/绝技20 | **瞬间美学**（写结果不写过程） | ✅ 可混搭 |
| 人物塑造 | 招牌+绝技 | **重复定型**（同一特征如音乐动机反复） | ✅ 可叠加 |
| 句式特征 | 短句+对仗 | **绝对化句式**（"绝无/任何/永远"） | ✅ 可调节 |
| 对话密度 | 40-55% | **50-70%**（纯对话页） | ✅ 可调节 |
| 哲学底色 | 轻松中的感悟 | **存在孤独**（寂寞是所有人的命运） | ✅ 可选色调 |
| 结构模式 | 起承转合 | **侦探推理结构**（谜面→线索→揭示） | ⚠️ 仅长篇可选 |

**使用方式**：
```
典雅白话 武侠短篇 古龙风：<灵感>      # 整体切换古龙模式
典雅白话 武侠中篇 古龙风：<灵感>
典雅白话 武侠长篇 古龙风：<灵感>

典雅白话 武侠中篇：<灵感> +古龙打斗 +古龙句式   # 仅启用部分技法
```

---

### 技法一：警句断言（替代反问叙事）

**原理**：古龙的叙述者不向读者提问，而是直接下判断。这创造了叙述者"什么都知道"的权威感，与反问叙事的"邀请共情"形成不同质感。

**操作规则**：
- 警句放在**段尾或场景切换处**——不与反问相同的位置规则
- 警句内容：关于**人生/江湖/人心的普遍性断言**
- 警句长度：不超过15字为佳
- 警句之后**不展开论证**——断言即止，切回场景

**句式模板**：
```
"人生本就……"
"这世上……的人，从来都……"
"一个人若……，就必定……"
"真正……的人，永远……"
"任何人……都……"
```

**密度控制**：
| 篇幅 | 警句上限 | 超出风险 |
|------|---------|---------|
| 短篇（8千-3万） | 3-5处 | 超过→说教感 |
| 中篇（3-10万） | 每章1-2处，全篇≤20处 | 超过→警察疲劳 |
| 长篇（10万+） | 每章1-2处 | 超过→风格自噬 |

**与反问叙事的区别示范**：

反问叙事（白话江湖风）：
> 三人一路乘着春风暖日，走向春风得意楼。好心办坏事岂非是生活中比较常见的事？

警句断言（古龙风）：
> 三人一路乘着春风暖日，走向春风得意楼。
>
> 好心办坏事的人，从来都不觉得自己做错了什么。

**混搭规则**：同一章中，反问和警句可以混用，但不能连续出现。两者之间至少间隔500字。反问结尾用"？"，警句结尾用"。"——前者邀请思考，后者不容置疑。

---

### 技法二：瞬间美学（替代快招速写/绝技慢写）

**原理**：古龙写战斗不写过程——写结果。读者看不到"剑如何刺出"，只看到"剑已刺入"。这不是偷懒，是刻意制造的神秘感和震撼力。

**三级写法**：

| 级别 | 写法 | 何时用 | 示例 |
|------|------|--------|------|
| **光速** | 只写结果，不写过程 | 角色实力碾压/展示高手差距 | "没有人看清他是怎么出手的。" |
| **一瞬** | 写一个决定性动作 | 均势对决的关键一击 | "他的剑忽然就到了对手的咽喉前。" |
| **凝时** | 慢写一个瞬间（≤200字） | 全书最重要的1-2场对决 | 可逐帧写，但只写**一刻**，不写整套招式 |

**光速级铁则**：
- 用否定句建立神秘感："没有人看清……""谁也不知道……""直到血流出，才有人发现……"
- 用旁观者反应代替打斗描写："每个人的眼睛都瞧直了" "两条腿不停地弹琵琶"
- **绝不在同一场打斗中既用"光速"又用"凝时"**——风格不统一，读者会出戏

**与现有打斗体系的互操作**：
- 古龙模式默认使用瞬间美学
- 但可以只对**高手角色**启用瞬间美学，杂兵战斗仍用快招速写——形成武力层级的叙事差异
- 长篇的终极对决可以使用古龙式"凝时"——整场战斗只慢写一个瞬间，其余依然是快招概括

---

### 技法三：重复定型（叠加角色系统）

**原理**：古龙的人物有一个容易辨认的"定型动作"或"定型特征"，每次出场都重复它。重复不是缺点，是音乐般的主题动机。

**操作规则**：
- 每个主要角色锁定**1-2个定型**（再多就散了）
- 定型在角色**首次出场**时建立
- 此后每个重要场景，至少让定型**出现一次**
- 定型在**关键情绪时刻**可以打破——打破本身就构成巨大的叙事力量

**定型类型与示例**：

| 定型类型 | 古龙原例 | 可效仿的示例 |
|---------|---------|------------|
| **动作定型** | 李寻欢刻木像 | 他每次落座之前，先用袖子拂一下椅子——即使椅子是干净的 |
| **语言定型** | 阿飞说话从来不解释 | 她说任何话之前，都先说一个"嗯"——不长不短，刚好一秒 |
| **物件定型** | 陆小凤摸胡子 | 他腰间挂了一个从不离身的旧葫芦——里面装的是水，不是酒 |
| **状态定型** | 李寻欢不停地咳嗽 | 他的左眼总是在跳——不是紧张，是旧伤 |
| **饮食定型** | 楚留香摸鼻子 | 他只喝一种茶——过了三泡就倒掉，说"没味道了" |

**重复的频率控制**：
| 篇幅 | 建议重复次数 | 过犹不及 |
|------|------------|---------|
| 短篇 | 3-5次 | 超过→机械 |
| 中篇 | 每章1-2次 | 超过→烦人 |
| 长篇 | 每1-2章出现一次 | 连续3章未提→读者忘了定型 |

**打破定型的时刻**——这是最有力的叙事武器：
> 李寻欢的咳嗽停了。不是好了——是他忘记咳了。忘记咳的这一刻，才是最可怕的时刻。
- 定型被打破 = 角色被某件事彻底改变
- 一篇小说中，同一个定型的打破建议不超过1次（长篇每卷1次）

**与现有角色系统的关系**：重复定型叠加在现有"角色绝技/招牌"之上。招牌告诉读者"这人有什么"，定型告诉读者"这人每次出现都会做什么"。

### 技法四：绝对化句式

**原理**：古龙的语言特征不仅在内容，更在**句式选择**。绝对化句式赋予叙述者一种不容置疑的权威。

**核心词汇**：
```
"绝" —— 绝无、绝不、绝未、绝不可能
"任何" —— 任何人、任何事、任何时候、任何地方
"永远" —— 永远不、永远是、永远无法
"从未" —— 从未见过、从未有人、从未想过
"一定" —— 一定不会、一定会、一定是
"就是" —— 这就是、那就是、本就是
```

**使用规则**：
- 绝对化词汇用于**叙述者的判断**，不用于客观事实描述
- 每千字中绝对化句式控制在**3-6句**（太少→没有古龙味；太多→审美疲劳）
- 绝对化句式不用于角色对话（除非这个角色的性格就是"说话绝对"）
- **同一句中不超过一个绝对词**——"任何人绝不可能永远……"是坏的模仿

**有效与无效的绝对化**：

✅ 有效（叙述者判断）：
> "这世上绝没有一个少女能抵挡江枫的微微一笑。"
> ——判断，作者说了算。读者不服也得服。

❌ 无效（客观描述用绝对词）：
> "他绝对是用剑刺中了对手的肩膀。"
> ——这是客观事实，用"绝对"反而显得叙述者不确定。

---

### 技法五：纯对话叙事

**原理**：古龙将对话占比推到极限。整页可以没有一句叙述，全是对话。叙述退化为极简的舞台提示。

**操作规则**：
- 两人对话时，**连续6-8个回合不带任何标签**（"他说""她道"全删）
- 超过8个回合插入一行**纯动作提示**（不做描述，只写动作）
- 一个场景的最高密度：**连续20+回合的对话**，中间只有1-2行动作

**纯对话示例**：
```
"你是聋子？"
"我喝不起。"
"我请你喝酒，用不着你花钱买。"
"不是我自己买来的东西，我绝不要。"
"我的话已经说得够清楚了么？"
"够清楚了。"
"好，你走吧。"
"好，我请你。"
```

**什么时候用纯对话**：
- ✅ 两个角色初次见面、互相试探
- ✅ 关键信息的揭露（说出来的比写出来的有力）
- ✅ 对决前的对峙（对话就是战斗的前奏）
- ❌ 群戏（3人以上对话需要标签区分谁在说）
- ❌ 需要同时展示角色内心活动的场景

**纯对话的节奏控制**：
- 每段纯对话之后，用一行**描绘环境或动作的短句**作为呼吸——"风从窗外吹进来。" "酒已凉了。"
- 这行短句的作用是让读者喘一口气，然后继续

### 技法六：侦探推理结构（长篇武侠可选）

**原理**：古龙将侦探小说的结构引入武侠。主线不是"主角成长变强"，而是"解开一个谜"。

**适用条件**：
- ⚠️ 仅建议中篇和长篇使用——短篇没有空间铺设线索
- 中篇：一个核心谜题贯穿全篇
- 长篇：可以每卷一个谜题 + 一个贯穿全书的终极谜题

**结构模板**：

```
谜面建立（第1-3章）
  → 一个不可能的事件发生了（密室/不可能犯罪/身份谜/武功谜）
  → 主角被卷入（可以是主动调查/被怀疑/受委托）
线索收集（第4-8章）
  → 每章发现一个新线索，同时排除一种可能性
  → 线索的来源多样化：对话/物证/武功痕迹/伤口特征
第一次推理（第8-10章）
  → 主角给出一个解释——但解释有破绽
  → 这个"错误的推理"导致更大的危机
转折揭示（第11-15章）
  → 真正的罪犯/真相被揭示
  → 揭示的方式必须是读者重读前文能发现的——不是新信息，是旧信息的重新理解
```

**侦探结构的武侠化**：
- 线索要用**武功**来呈现：伤口的形状暗示了凶手的武功路数、现场残留的内力痕迹、独门暗器的特征
- 推理要用**江湖逻辑**：不是法庭证据，而是"什么样的武功才能做到这一点"
- 结局要有**动作释放**：推理完毕→战斗。不能"说破就结束"

**不适合用侦探结构的情况**：
- 主角成长线为主的成长小说
- 大规模的江湖争霸/门派战争
- 读者已知真相、看角色如何发现的倒叙结构（那应该用"读者全知"视角）

### 技法七：存在孤独底色

**原理**：古龙小说的深处，总有一种"人从根本上就是孤独的"哲学底色。这不是情节——是调性。

**操作方式**（这是最轻的技法，不改变结构，只改变叙述的语气）：
- 在场景转换时，**偶尔**让叙述者落一句关于孤独的轻叹
- 在角色独处时，**不动声色**地写他的安静——不是内心独白，是外部行为
- **最重要的原则**：不说"他很孤独"。写他在热闹的酒楼里一个人喝茶，茶凉了也不知道

**示例**：
> 雪，终于停了，天地间的寒气却更重，寂寞也更浓。
>
> ——这是第一页就定下的调子。不是"他很寂寞"，是"寂寞也更浓"。寂寞被写成了一种天气。

**何时使用**：
- 故事的基调偏冷、偏暗、偏个体时——启用
- 故事基调偏暖、偏群像、偏欢乐时——不启用（古龙的《欢乐英雄》就没有孤独底色）
- 可以仅对**某一个角色**启用孤独底色——其他角色正常

### 古龙模式的常见失败模仿

| 失败模式 | 症状 | 修复 |
|---------|------|------|
| **警句通货膨胀** | 每段结尾都来一句"人生本就……" | 警句不是标点符号。只在叙述者"忍不住想说"的时候说。 |
| **假古龙式断句** | 把所有长句都拆成短句 | 短句是**节奏感**，不是机械拆分。连续短句之后必须有一个长句"稳住"。 |
| **学古龙写酒写女人** | 每页都出现酒，每个女性都风情万种 | 古龙的酒和女人是**他本人的嗜好**，不是"古龙风"的技法核心。学他如何用酒推动情节，不是学他写酒。 |
| **绝对化过度** | "绝对""任何"每句话都有 | 绝对化词汇的效果来自**稀有**。如果每句都用，叙述者就从"权威"变成了"偏执"。 |
| **光速打斗滥用** | 任何战斗都不写过程 | 档次拉开：杂兵→光速，高手→一瞬，终战→凝时。全用光速等于谁都不厉害。 |
| **假装有深度** | 让角色莫名其妙地说深沉的话 | 古龙的角色说哲理是因为**他们的处境**让他们不得不说。不是为了"显得深刻"。 |
| **破折号过量** | 密度 > 8/k，把破折号当万能标点 | 详见 `dianya-novel-engine` 破折号使用标准。仅三类场景可用（截断语/警句延展/主题揭示），目标密度 2~7/k。 |

  → 每章写作：写前强制5步 → 写作 → 写后强制4步（中长篇）
